一、 什么是字符串？

知识讲解：

同学们，我们之前在 print() 函数里输出过文字，比如 "你好，世界！"。在Python中，像这样一串文字、字母、数字或者符号，我们都称之为字符串 (String)。

你可以把字符串想象成一串用绳子串起来的珠子，每个珠子就是一个字符（比如一个汉字、一个英文字母、一个数字或者一个标点符号）。

字符串是编程中非常常用的一种数据类型，用来表示文本信息。比如，你的名字、一本书的标题、一条消息等等，都可以用字符串来表示。

在Python中，字符串的数据类型缩写是 str。

思考与讨论：

  • 除了我们之前打印过的"你好，世界！"，你还能想到哪些信息可以用字符串来表示吗？（例如：你的学校名称、你最喜欢的颜色、一句歌词等）
  • 数字 123 和字符串 "123" 有什么不同呢？（提示：想一想它们能不能直接做数学加法）

二、 创建字符串：给文字穿上"引号"外衣

知识讲解：

为了让Python知道我们写的是一串文字（字符串），而不是一个变量名或者一个数字，我们需要给这些文字"穿上外衣"——也就是用引号把它们包围起来。

Python提供了两种常用的引号来创建字符串：

  • 单引号 (' ')： 
    my_pet = '小猫'
print(my_pet)
print(type(my_pet)) # 会输出 <class 'str'>
  • 双引号 (" ")： 
    my_dream = "成为宇航员"
print(my_dream)
print(type(my_dream)) # 也会输出 <class 'str'>

单引号和双引号是等效的：用单引号创建的字符串和用双引号创建的字符串，对于Python来说是一样的，没有区别。你可以根据自己的喜好或者具体情况来选择使用哪一种。

什么时候需要特别选择？

  • 如果你的字符串本身就包含单引号，那么最好用双引号把它包起来，例如： message = "I'm a student."
  • 如果你的字符串本身就包含双引号，那么最好用单引号把它包起来，例如： book_title = '他说："你好！"'
  • 如果字符串里既有单引号又有双引号，那怎么办呢？后面我们会学到一种叫做"转义字符"的东西来解决。

三、 字符串的"魔法"操作：连接与重复

知识讲解：

字符串除了可以存储文字，还可以进行一些有趣的"魔法"操作。

字符串连接 (Concatenation) - 用加号 +：

  • 我们可以用加号 + 把两个或多个字符串"粘"在一起，形成一个新的、更长的字符串。 
    str1 = "Python"
str2 = "真棒"
result = str1 + " " + str2  # 中间加一个空格字符串
print(result)           # 会输出：Python 真棒

字符串重复 (Repetition) - 用乘号 *：

  • 我们可以用乘号 * 把一个字符串重复好几次。你需要一个字符串和一个整数。 
    separator = "-" * 10  # 把字符串"-"重复10次
print(separator)      # 会输出：----------
greeting = "加油！" * 3
print(greeting)       # 会输出：加油！加油！加油！

练习题2 (写出 print() 语句的输出结果)：

1. print("Go" + "Fish")

答案与解析：

答案： GoFish

解析： 两个字符串直接连接在一起。

2. print("Ha" * 4)

答案与解析：

答案： HaHaHaHa

解析： 字符串 "Ha" 重复了4次。

3. 

part1 = "Python is "
part2 = "fun!"
print(part1 + part2)
答案与解析：

答案： Python is fun!

解析： 变量 part1 和 part2 存储的字符串被连接起来。

练习题3 (找出下面代码中的错误并改正)：

# 原始错误代码
count = 5
message = "我有 " + count + " 个苹果。"
print(message)
错误原因：

count 是一个整数变量，不能直接用 + 与字符串 "我有 " 和 " 个苹果。" 连接。

改正后的代码：
count = 5
message = "我有 " + str(count) + " 个苹果。" # 将整数count转换为字符串
print(message)

四、 多行文字怎么办？三引号来帮忙！

知识讲解：

有时候，我们想表示一段比较长的文字，这段文字本身就分成了好几行，比如一首诗、一段歌词或者一个地址。

如果用普通的单引号或双引号，我们就得在每行的末尾加上一个特殊的换行符 \n，会比较麻烦。

Python提供了一种更方便的方式来创建多行字符串：使用三个单引号 ''' '''或者三个双引号 """ """把整段文字包起来。

在三引号里面的文字，你写的时候是什么样（包括换行和空格），它存到字符串里就是什么样。

poem = """静夜思
床前明月光，
疑是地上霜。
举头望明月，
低头思故乡。"""
print(poem)

address = '''北京市
海淀区
中关村大街1号'''
print(address)

还记得吗？三引号除了可以创建多行字符串，如果它没有被赋值给任何变量，也可以当作多行注释来用。

本课总结回顾

今天我们深入学习了字符串这个"文字"类型的数据。我们知道了怎么用单引号或双引号来创建它们，还学会了用加号 + 把字符串"粘"起来，用乘号 * 让字符串"变多"。最重要的是，我们明白了字符串和数字是不同的"物种"，不能随便混在一起运算，需要的时候要用 str() 这样的函数来"变身"。最后，我们还掌握了用三引号轻松创建多行文字的方法。字符串在编程中无处不在，熟练掌握它对我们非常重要！
