一、 程序的"三大件"：输入、处理、输出

知识讲解：

同学们，我们写的每一个小程序，不管多简单或多复杂，通常都可以看作是由三个主要部分组成的，就像我们做饭有买菜（输入）、洗菜切菜炒菜（处理）、上菜（输出）一样。

  • 输入 (Input)：就是程序从外部获取信息的过程。比如，让用户从键盘输入一个数字，或者程序从一个文件中读取数据。
  • 处理 (Processing)：这是程序的核心部分，程序会根据我们写的指令对输入的信息进行计算、判断、转换等操作。
  • 输出 (Output)：就是程序把处理的结果展示出来的过程。比如，在屏幕上显示一句话、一个数字，或者把结果保存到文件中。

今天我们主要学习"输出"，也就是怎么让电脑把我们想显示的东西显示出来。

思考与讨论：

  • 如果我们想写一个程序来计算两个数字的和，那么这个程序的"输入"、"处理"和"输出"分别是什么呢？

二、 Python 代码的基本样子：格式很重要

知识讲解：

写 Python 代码就像写作文一样，也有一些格式要求，电脑才能正确理解。

缩进 (Indentation)：

  • 这是 Python 非常非常重要的一个特点！不像其他一些编程语言用括号或者特定的词来区分代码块，Python 用空格或者制表符(Tab键)产生的"缩进"（就是代码行开头的空白）来表示哪些代码是一组的，或者说哪些代码是属于哪个部分的。
  • 通常，我们用 4个空格作为一个缩进级别。
  • 如果缩进不对，程序就会出错，或者运行的结果不是我们想要的。我们会在后面的课程中更详细地学习什么时候需要缩进。现在大家先记住，Python 对开头的空格很"敏感"。

注释 (Comments)：

  • 注释是写在代码中，但电脑不会执行的部分。它们是给我们自己或者其他看代码的人看的，用来解释这段代码是做什么的，或者为什么要这么写。好的注释能让代码更容易懂。
  • 单行注释：以 # 号开始。从 # 号到这一行的末尾，都算是注释。 
    # 这是一行单行注释，电脑会忽略它
  • 多行注释：可以用三个单引号 ''' 或者三个双引号 """ 把一大段文字包起来，形成多行注释。 
    
"""
这是
一段多行
注释。
"""

'''
这也是
一段多行
注释。
'''
    注意：在之前的试卷中，有提到 ### 不是多行注释的正确方式，大家要记住哦！

字符串标识 (String Identification)：

  • 我们在上一节课用 print("你好，世界！") 的时候， "你好，世界！" 就是一个字符串，也就是一串文字。
  • 在 Python 中，字符串需要用单引号 ' ' 或者双引号 " " 包围起来，电脑才知道这是一段文字。例如：'你好' 和 "你好" 都是正确的字符串。

(新补充) 语句分隔符 (Statement Separator)：

  • 通常，我们一行只写一条 Python 语句（一个完整的指令）。
  • 但如果需要在同一行写多条语句，可以用分号 ; 把它们隔开。不过，为了代码清晰易读，一般不推荐在一行写太多语句。 
    print("第一句"); print("第二句") # 不太推荐这样写，但语法是允许的

练习题1 (判断对错)：

1. Python 代码的缩进只是为了好看，不影响程序运行。

2. 以 # 开头的行是 Python 的单行注释。

3. 在 Python 中，可以在一行内写多条语句，用逗号 , 分隔它们。

三、 超级有用的 print() 函数：让电脑"说话"

知识讲解：

我们已经用过 print() 函数来输出文字了。print() 是 Python 中一个非常基础也非常重要的内置函数，它的主要工作就是把我们想要显示的内容输出到屏幕（也叫控制台）上。

print() 的基本用法：

  • 输出字符串：把想输出的文字用引号包起来放在括号里。 
    print("我爱Python！")
print('Python真有趣！')
  • 输出数字：可以直接把数字放在括号里。 
    print(123)
print(3.14)
  • 输出表达式的计算结果：print() 可以直接计算括号里的数学表达式，并输出结果。 
    print(10 + 5)  # 会输出 15
print(3 * 4)   # 会输出 12
  • 如果我们把数学表达式用引号包起来，print() 就会把它当作字符串原样输出，而不会计算。 
    print("10 + 5") # 会输出 10 + 5
  • 输出多个内容：可以在 print() 的括号里放多个东西，用逗号 , 把它们隔开。输出的时候，这些内容之间默认会用一个空格隔开。 
    name = "小明"
age = 8
print(name, "今年", age, "岁了。") # 会输出：小明 今年 8 岁了。
  • 默认换行：每次调用 print() 函数，输出完内容后，它默认会在末尾加上一个"换行符"，所以下一次 print() 的内容会显示在新的一行。 
    print("第一行")
print("第二行")
# 输出会是：
# 第一行
# 第二行
  • 控制结尾不换行 end 参数：如果我们不希望 print() 输出后换行，而是想让下一次的输出紧跟在后面，我们可以给 print() 函数传递一个特别的参数 end。
    end 参数用来指定 print() 输出结束后用什么字符结尾，默认是换行符 \n。
    如果我们想让它用空格结尾，可以写 end=' '。如果想什么都不加，就写 end=''。 
    print("你好，", end="") # 输出后不换行，也不加空格
print("世界！")      # 这句会紧跟在"你好，"后面
# 输出会是：你好，世界！

print("苹果", end="...") # 输出后加 "..." 而不是换行
print("香蕉")
# 输出会是：苹果...香蕉
  • 使用三引号输出多行文本：如果想输出一大段包含换行的文字，可以直接用三个单引号 ''' ''' 或三个双引号 """ """ 把文字包起来，放在 print() 函数里。 
    print("""
静夜思
床前明月光，
疑是地上霜。
""")

四、 认识 type() 函数：查看数据的"身份证"

知识讲解：

我们在Python中处理的各种东西，比如数字 123、文字 "你好"，它们在Python里都有自己的"身份"，叫做数据类型 (Data Type)。

想知道一个数据或者一个变量里存的是什么类型的数据，我们可以用 type() 函数来看它的"身份证"。

把你想查看的东西放在 type() 的括号里，然后用 print() 把结果显示出来。

print(type(100))       # 100 是整数，会输出 
print(type(3.14))      # 3.14 是小数（浮点数），会输出 
print(type("你好"))    # "你好" 是文字（字符串），会输出 
print(type(True))      # True 是表示"真"的值（布尔型），会输出

即使是像 "int" 这样一个看起来像类型名字的文字，如果它被引号包起来了，那它本身也是一个字符串。

print(type("int"))     # "int" 是字符串，会输出

练习题3 (写出输出结果)：

1. print(type(0.5))

答案与解析：

答案： <class 'float'>

解析： 0.5 是一个带小数的数字，属于浮点数类型。

2. print(type("Python123"))

答案与解析：

答案： <class 'str'>

解析： 用引号包围起来的都是字符串，即使里面包含数字。

3. 

x = 2025
print(type(x))
答案与解析：

答案： <class 'int'>

解析： 变量 x 存储的是整数 2025。

本课总结回顾

今天我们学习了Python程序的基本样子，知道了缩进和注释的重要性。我们还重点练习了 print() 函数，学会了怎么让电脑输出各种信息，包括文字、数字、计算结果，以及怎么控制输出的结尾是不是换行。最后，我们还认识了 type() 函数，它可以帮我们看清数据到底是什么类型的。这些都是非常基础但非常重要的知识，打好基础，后面的学习就会更顺利！
