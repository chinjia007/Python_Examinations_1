一、 让程序"听"我们说话： input() 函数

知识讲解：

同学们，我们之前学习了用 print() 函数让电脑把信息"说"出来（输出）。现在，我们要学习怎么让电脑"听"我们说话，也就是从我们这里获取信息，这就是输入。

在Python中，我们用 input() 函数来获取用户从键盘输入的内容。

input() 函数的基本用法：

my_name = input()
print("你好，" + my_name)

当你运行这段代码时，程序会在 input() 这一行停下来，光标会闪烁，等待你在键盘上输入一些内容，然后按下回车键。你输入的内容就会被存到变量 my_name 中。

给用户友好提示：为了让用户知道应该输入什么，我们通常会在 input() 函数的括号里放一段提示文字（字符串）。这段提示文字会显示在屏幕上，引导用户输入。

user_age_str = input("请输入您的年龄：") # "请输入您的年龄：" 就是提示文字
print("您输入的年龄是：" + user_age_str)

用户通过 input() 函数输入的数据，并不会立刻直接显示到控制台的输出区域。它首先被程序"接收"并存储起来（通常是存到一个变量里）。如果想看到输入的内容，我们还是需要用 print() 函数把它打印出来。

思考与讨论：

  • 如果我们想写一个程序，让用户输入他最喜欢的颜色，我们应该怎么写这个 input() 语句的提示信息呢？
  • 用户输入完成后，按哪个键表示输入结束？ (回车键)

二、 input() 函数的"秘密"：它只认字符串！

知识讲解：

这是 input() 函数一个非常非常重要的特点，大家一定要牢牢记住：无论你从键盘输入的是数字、文字还是符号，input() 函数把它"接收"进来之后，都会把它当作字符串类型来处理和存储。

举个例子：

num_str = input("请输入一个数字：") # 假设你输入了 10
print(type(num_str))             # 你会看到输出是 <class 'str'>

即使你输入的是数字 10，变量 num_str 里存的也是字符串 '10'，而不是数字 10。

练习题1 (判断对错)：

1. 如果我用 input() 输入了数字 5，那么存到变量里的就是整数 5。

2. x = input("请输入你的名字：") 这行代码执行后，变量 x 的类型一定是字符串。

三、 "变身魔法"：数据类型转换

知识讲解：

既然 input() 获取的总是字符串，那如果我们想用用户输入的数字来进行数学计算（比如加减乘除）怎么办呢？直接用字符串 '10' 和数字 5 相加，程序会报错的！

这时候，我们就需要施展"变身魔法"，把字符串类型的数字"变回"真正的数字类型。这个过程叫做数据类型转换 (Type Casting)。

Python提供了几个常用的"变身"函数：

  • int() 函数：变身为整数
    • 作用：尝试将一个字符串或者一个浮点数转换成一个整数。
    • 示例： 
      age_str = input("请输入你的年龄：") # 输入 "8"
age_int = int(age_str)          # age_int 的值就是整数 8
print(age_int + 2)              # 可以正常计算，输出 10
    • int() 转换时的注意事项：
      • 如果转换的是一个浮点数，int() 会直接去掉小数部分（截断），而不是四舍五入。例如：int(7.8) 的结果是 7。
      • 如果转换的是一个字符串，这个字符串必须看起来像一个纯整数。如果字符串里有小数点（如 "7.8"）或者其他非数字字符（如 "abc"），用 int() 转换它就会报错 (ValueError)。例如：int("7.8") 会报错。
  • float() 函数：变身为浮点数（小数）
    • 作用：尝试将一个字符串或者一个整数转换成一个浮点数。
    • 示例： 
      price_str = input("请输入商品价格：") # 输入 "29.9"
price_float = float(price_str)      # price_float 的值就是浮点数 29.9
print(price_float * 0.9)          # 可以进行打九折计算

num_int = 100
num_float = float(num_int)          # num_float 的值是浮点数 100.0
print(num_float)
    • 注意：float() 可以转换只包含数字的字符串（如 "123" 会变成 123.0）或者包含小数点的数字字符串（如 "3.14" 会变成 3.14）。但如果字符串包含其他非数字字符，也会报错。
  • str() 函数：变身为字符串
    • 作用：将一个数字（整数或浮点数）或者其他类型的数据转换成一个字符串。
    • 这个我们之前在 print() 里连接字符串和数字时已经用过了。
    • 示例： 
      my_number = 10
message = "我有 " + str(my_number) + " 个气球。" # str(10) 变成 "10"
print(message)                                 # 输出: 我有 10 个气球。

score = 95.5
report = "你的得分是：" + str(score)
print(report)                                  # 输出: 你的得分是：95.5
    • 例如，str(49) 的结果就是字符串 '49'。

四、 eval() 函数简介 (了解即可)

知识讲解：

Python 还有一个函数叫 eval()。它的功能比较强大，可以把一个字符串当成一个有效的Python表达式来计算，并返回计算结果。

例如：

expression_str = "2 + 3 * 4"
result = eval(expression_str)
print(result)  # 会输出 14 (因为 3*4=12, 2+12=14)

num_as_str = "10.5"
num_val = eval(num_as_str)
print(type(num_val)) # 会输出 <class 'float'>，因为 "10.5" 被当作浮点数处理了

注意：虽然 eval() 看起来很方便，但它也有一些安全风险。如果 eval() 括号里的字符串是由用户输入的，而用户输入了一段有破坏性的代码，eval() 也会尝试执行它，这可能会导致程序出问题。因此，在很多情况下，尤其是在处理不完全信任的输入时，我们应该避免使用 eval()，而是选择更安全的 int() 或 float() 进行明确的类型转换。

在一些考试平台或特定环境中，eval() 函数可能会被禁用，就是出于安全考虑。

练习题3 (思考与讨论)：

1. 如果 my_string = "100"，那么 int(my_string) 和 eval(my_string) 的结果和类型一样吗？

答案与解析：

结果都是整数 100，类型都是 <class 'int'>。

2. 如果 my_string = "3.14"，那么 float(my_string) 和 eval(my_string) 的结果和类型一样吗？

答案与解析：

结果都是浮点数 3.14，类型都是 <class 'float'>。

3. 为什么通常情况下，用 int() 或 float() 进行转换比用 eval() 更安全？

答案与解析：

因为 int() 和 float() 只会尝试把字符串转成特定类型的数字，如果字符串内容不符合数字格式就会报错。而 eval() 会尝试执行字符串里的任何Python代码，如果字符串是用户输入的，可能会有安全隐患。

本课总结回顾

今天我们学会了如何使用 input() 函数从用户那里获取信息，并且知道了 input() 返回的总是字符串。更重要的是，我们掌握了 int()、float() 和 str() 这三个"变身魔法"函数，它们可以帮助我们在不同的数据类型之间进行转换，这对于处理用户输入并进行计算至关重要。我们还简单了解了 eval() 函数，并知道了在什么时候应该谨慎使用它。能够正确地处理输入和转换数据类型，是编写交互式程序的关键一步！
