一、 电脑也会算术：Python的算术运算符

知识讲解：

同学们，我们平时在数学课上会做加减乘除，电脑用Python也可以帮我们做这些计算，而且速度非常快，还不容易出错！Python提供了一些特殊的符号，叫做算术运算符 (Arithmetic Operators)，来执行这些数学运算。

我们来认识一下它们：

  • 加法 (+)：和我们数学里的一样，用来计算两个数相加。 
    result = 10 + 5
print(result)  # 输出：15
  • 减法 (-)：用来计算一个数减去另一个数。 
    result = 10 - 5
print(result)  # 输出：5
  • 乘法 (*)：用来计算两个数相乘。 Python中3乘以5写作 3*5) 
    result = 10 * 5
print(result)  # 输出：50
  • 除法 (/)：用来计算一个数除以另一个数。 特别注意：在Python中，/ 进行的除法运算，即使两个数都是整数并且能整除，结果也总是浮点数（带小数点的数）。 
    result1 = 10 / 5 
print(result1) # 输出：2.0 
result2 = 7 / 2
print(result2) # 输出：3.5
  • 整除 (//)：这个运算符只保留除法结果的整数部分，把小数部分去掉（不是四舍五入，是直接去掉）。 
    result1 = 10 // 5 
print(result1) # 输出：2 
result2 = 7 // 2 
print(result2) # 输出：3 (因为7除以2等于3余1，整数部分是3) 
result3 = 12 // 5 # 12除以5等于2余2
print(result3) # 输出：2
  • 取余 (取模) (%)：这个运算符计算的是除法的余数。 
    result1 = 10 % 3 # 10除以3等于3余1 
print(result1)   # 输出：1 
result2 = 7 % 2  # 7除以2等于3余1 
print(result2)   # 输出：1 
result3 = 123 % 100 # 123除以100等于1余23
print(result3)    # 输出：23
  • 幂运算 (**)：用来计算一个数的多少次方。比如 a ** b 就是计算 a 的 b 次方。 
    result1 = 2 ** 3  # 2的3次方，即 2 * 2 * 2 
print(result1)    # 输出：8 
result2 = 10 ** 2 # 10的2次方，即 10 * 10
print(result2)    # 输出：100

二、 先算哪个？运算的顺序很重要！

知识讲解：

当一个数学式子里有很多不同的运算符时，Python会按照一定的顺序来进行计算，这个顺序叫做运算优先级 (Operator Precedence)。这和我们数学课上学的"先乘除后加减，有括号先算括号里的"规则很像。

Python中的运算优先级（从高到低）：

  1. 括号 ()：括号里的运算最先进行。
  2. 幂运算 **：比乘除法优先级高。
  3. 乘法 *、除法 /、整除 //、取余 %：这些运算符优先级相同。如果它们同时出现，一般会从左到右依次计算。
  4. 加法 +、减法 -：这些运算符优先级相同，也比上面的低。如果它们同时出现，也是从左到右计算。

举例：

result = 5 + 3 * 2
# 先算乘法 3 * 2 = 6
# 再算加法 5 + 6 = 11
print(result) # 输出：11

result_with_brackets = (5 + 3) * 2
# 先算括号里的 5 + 3 = 8
# 再算乘法 8 * 2 = 16
print(result_with_brackets) # 输出：16

# 幂运算优先于乘法
print(10 * 2 ** 3) # 先算 2**3 = 8，再算 10 * 8 = 80

# 比较运算符优先级低于算术运算符，逻辑运算符优先级最低
# 例如： 3*5 > 2 and 6 >= 6.0，会先算 3*5

三、 写代码更简洁：复合赋值运算符

知识讲解：

在编程中，我们经常会遇到这样的情况：把一个变量原来的值取出来，进行一些运算，然后再把结果存回到这个变量自己身上。

比如，我们要给变量 score 加上10分，可能会写： score = score + 10。

Python提供了一种更简洁的写法，叫做复合赋值运算符 (Compound Assignment Operators)。

常见的复合赋值运算符：

  • +=：加法赋值 (例如 x += 5 等价于 x = x + 5)
  • -=：减法赋值 (例如 y -= 3 等价于 y = y - 3)
  • *=：乘法赋值 (例如 z *= 2 等价于 z = z * 2)
  • /=：除法赋值 (例如 w /= 4 等价于 w = w / 4)
  • //=：整除赋值 (例如 u //= 2 等价于 u = u // 2)
  • %=：取余赋值 (例如 v %= 3 等价于 v = v % 3)
  • **=：幂赋值 (例如 p **= 2 等价于 p = p ** 2)

例如题目中的 a*=a-b 就等价于 a = a * (a-b)。要注意 a-b 会先被计算。

使用复合赋值运算符可以让我们的代码看起来更简短，也更容易读懂一些常见的操作。

答案：

答案：

四、 用数学解决问题：编程实践

知识讲解：

我们学习了这么多算术运算符，就是为了能用程序来帮助我们解决生活中的数学问题。

比如计算购物总价、计算平均分、或者像之前题目里计算小孩排队、计算正方形周长、计算两数的和与积的乘积等等。

关键在于把问题分解成一步一步的计算，然后用Python代码把这些计算步骤写出来。

编程题1 (在IDLE中完成)：

题目：小明有20颗糖果，他想平均分给他的3个好朋友，剩下的糖果自己留着。请编写一个程序，计算每个好朋友能分到多少颗糖果，以及小明自己能剩下多少颗糖果。

提示：你需要用到整除 // 和取余 % 运算符。

参考答案：
total_candies = 20
num_friends = 3

candies_per_friend = total_candies // num_friends
remaining_candies_for_ming = total_candies % num_friends

print("每个好朋友可以分到：", candies_per_friend, "颗糖果")
print("小明自己剩下：", remaining_candies_for_ming, "颗糖果")
解析：
  • total_candies // num_friends 计算了每人分到的整数颗糖。
  • total_candies % num_friends 计算了分完后剩下的糖果数。

编程题2 (在IDLE中完成)：

题目：一个长方形的操场，长是85米，宽是40米。请编写一个程序计算这个操场的周长和面积。

公式提示：
长方形周长 = (长 + 宽) * 2
长方形面积 = 长 * 宽

参考答案：
length = 85
width = 40

perimeter = (length + width) * 2
area = length * width

print("操场的周长是：", perimeter, "米")
print("操场的面积是：", area, "平方米")
解析：
  • 程序定义了长和宽。
  • 根据公式分别计算了周长和面积。
  • 使用 print() 输出了结果。
