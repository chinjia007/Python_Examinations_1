一、 对与错：认识布尔值

知识讲解：

同学们，在生活中，我们经常会判断一件事情是对的还是错的，是真的还是假的。比如，"太阳从东边升起"这是对的（真的），"鱼儿能在天上飞"这是错的（假的）。

在Python编程中，也有类似表示"对"与"错"、"真"与"假"的值，我们称它们为布尔值 (Boolean values)。

布尔值非常简单，它只有两种可能的值：

  • True：表示"真"、"对"、"成立"。
  • False：表示"假"、"错"、"不成立"。

注意：在Python中，True 和 False 这两个单词的首字母必须大写，它们是Python的关键字。

布尔值在编程中非常重要，它们是进行逻辑判断和控制程序流程的基础。

思考与讨论：

  • 请你说出生活中的一个"真"的描述和一个"假"的描述。
  • 如果我们想用布尔值来表示"今天是不是晴天？"，如果今天是晴天，应该用哪个布尔值？如果不是晴天呢？

二、 比一比大小：比较运算符

知识讲解：

我们经常需要比较两个东西的大小、或者判断它们是不是一样。比如，比较两个小朋友谁的个子高，或者看看两个苹果是不是一样重。

Python提供了一些特殊的符号，叫做比较运算符 (Comparison Operators)，来帮助我们进行这种比较。比较的结果，就会是一个布尔值 (True 或 False)。

常用的比较运算符：

  • 等于 (==)：
    • 用来判断左右两边的值是否相等。
    • 注意：是两个等号 ==，不是我们之前学的赋值用的一个等号 =！
    • 示例： 
      print(5 == 5)  # 5等于5吗？是的，所以输出：True
print(10 == 7) # 10等于7吗？不是，所以输出：False
print("hello" == "hello") # 字符串"hello"等于"hello"吗？是的，输出：True
print("Apple" == "apple") # 字符串"Apple"等于"apple"吗？不是，因为大小写不同，输出：False
  • 不等于 (!=)：
    • 用来判断左右两边的值是否不相等。
    • 示例： 
      print(5 != 5)  # 5不等于5吗？不是的（它们是相等的），所以输出：False
print(10 != 7) # 10不等于7吗？是的，所以输出：True
  • 大于 (>)：
    • 用来判断左边的值是否大于右边的值。
    • 示例： 
      print(10 > 5)  # 10大于5吗？是的，输出：True
print(3 > 8)   # 3大于8吗？不是，输出：False
  • 小于 (<)：
    • 用来判断左边的值是否小于右边的值。
    • 示例： 
      print(10 < 5)  # 10小于5吗？不是，输出：False
print(3 < 8)   # 3小于8吗？是的，输出：True
  • 大于等于 (>=)：
    • 用来判断左边的值是否大于或者等于右边的值。
    • 示例： 
      print(10 >= 5) # 10大于等于5吗？是的（大于），输出：True
print(5 >= 5)  # 5大于等于5吗？是的（等于），输出：True
print(3 >= 8)  # 3大于等于8吗？不是，输出：False
  • 小于等于 (<=)：
    • 用来判断左边的值是否小于或者等于右边的值。
    • 示例： 
      print(10 <= 5) # 10小于等于5吗？不是，输出：False
print(5 <= 5)  # 5小于等于5吗？是的（等于），输出：True
print(3 <= 8)  # 3小于等于8吗？是的（小于），输出：True

三、 True、False 与数字的"亲戚关系"

知识讲解：

在Python中，布尔值 True 和 False 有时候可以像数字一样参与运算或比较。

  • True 在数值上可以看作是 1。
  • False 在数值上可以看作是 0。

这种特性在某些情况下很有用，但我们初学时主要理解它们在比较时的行为。

示例：

print(True == 1)  # 输出：True (因为True相当于1)
print(False == 0) # 输出：True (因为False相当于0)
print(True > False)# 输出：True (因为1 > 0)
print(True + True) # 输出：2 (因为1 + 1 = 2)

例如之前试卷中出现 print(a==A)，如果 a=1，A=True，结果就是 True。

1. print(False != 0)

答案与解析：

答案： False

解析： False 等于 0，所以 False != 0 是 False。

2. print(True > 0)

答案与解析：

答案： True

解析： True 等于 1，1 > 0 是 True。

3.

print(result == 1)
答案与解析：

答案： True

解析： result 是 True，True == 1 是 True。

四、 用比较来做判断：编写简单的逻辑表达式

知识讲解：

我们现在可以用比较运算符来写一些简单的逻辑表达式 (Logical Expressions)。逻辑表达式就是那些计算结果为 True 或 False 的式子。

比如，我们要判断一个小朋友的年龄 child_age 是否达到了可以玩某个游戏的最低年龄 min_age，我们就可以写：can_play = child_age >= min_age。这个 can_play 变量就会存下 True (如果达到了) 或者 False (如果没达到)。

这种简单的判断是我们后面学习更复杂的条件控制（比如 if 语句）的基础。

编程题1 (在IDLE中完成)：

题目：身高检查器
编写一个程序，首先提示用户输入他们的身高（单位：厘米），例如："请输入您的身高（厘米）："。 将输入的身高转换为整数。 判断用户身高是否大于或等于120厘米。 输出一个布尔值，表示是否符合要求（例如，如果身高130，则输出 True）。

参考答案：
height_str = input("请输入您的身高（厘米）：")
height_cm = int(height_str)
is_tall_enough = height_cm >= 120
print("身高是否符合要求（>=120cm）：", is_tall_enough)
解析：
  • 程序获取输入并转为整数。
  • 使用 >= 比较身高和120。
  • 将比较结果（布尔值）存入变量并输出。

编程题2 (在IDLE中完成)：

题目：数字相等判断器
编写一个程序，提示用户连续输入两个整数。 判断这两个整数是否相等。 输出一个布尔值，表示它们是否相等。

参考答案：
num1_str = input("请输入第一个整数：")
num1 = int(num1_str)
num2_str = input("请输入第二个整数：")
num2 = int(num2_str)
are_equal = (num1 == num2) # 括号可以不加，但加上更清晰
print("两个数字是否相等：", are_equal)
解析：
  • 程序获取两次输入并都转为整数。
  • 使用 == 判断两个整数是否相等。
  • 输出比较结果。

本课总结回顾

今天我们认识了编程世界里的"对"与"错"——布尔值 True 和 False。我们还学会了使用各种比较运算符 (==, !=, >, <, >=, <=) 来比较数字和文字的大小或者是否相等，并且知道了这些比较的结果就是布尔值。我们还发现 True 和 False 有时候可以像数字 1 和 0 一样。这些比较判断的能力，是我们以后让程序变得更"聪明"、能根据不同情况做不同事情的关键哦！
