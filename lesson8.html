一、 连接"对"与"错"：认识逻辑运算符

知识讲解：

同学们，上一节课我们学习了比较运算符，它可以告诉我们一个条件是 True (对的) 还是 False (错的)。但有时候，我们需要判断多个条件的组合情况。比如，"如果今天是晴天 并且 我完成了作业，那么我就可以出去玩。"这里的"并且"就是一种逻辑连接。

Python提供了逻辑运算符 (Logical Operators)来帮助我们连接或修改布尔值和逻辑表达式。

主要的逻辑运算符有三个：

  • and (与/并且)：
    • 当用 and 连接两个条件时，只有两个条件都为 True，整个表达式的结果才是 True。只要有一个条件是 False，结果就是 False。
    • 就像开锁需要两把钥匙都对才能打开一样。
  • or (或/或者)：
    • 当用 or 连接两个条件时，只要至少有一个条件为 True，整个表达式的结果就是 True。只有两个条件都为 False，结果才是 False。
    • 就像你有两张电影票，一张是《功夫熊猫》，一张是《哪吒》，只要你有其中一张（或者两张都有），你就可以去看电影。
  • not (非/不是)：
    • not 运算符只作用于一个条件，它的作用是取反。如果原来的条件是 True，not 之后就变成 False；如果原来是 False，not 之后就变成 True。
    • 就像说"不是晴天"，如果"晴天"是真的，那么"不是晴天"就是假的。

思考与讨论：

  • "如果明天下雨 或者 明天刮大风，我就不去公园了。"这句话里用到了哪个逻辑连接词？这种情况下，什么条件下我才会去公园？
  • "这个苹果不是红色的。" 如果这个苹果真的是红色的，那么这句话是真的还是假的？

二、 逻辑运算符的"真值表"与运算规则

知识讲解：

我们可以用一个表格（叫做真值表）来清晰地表示逻辑运算符在不同情况下的结果：

and (与)

条件1 (A)条件2 (B)A and B
TrueTrueTrue
TrueFalseFalse
FalseTrueFalse
FalseFalseFalse

or (或)

条件1 (A)条件2 (B)A or B
TrueTrueTrue
TrueFalseTrue
FalseTrueTrue
FalseFalseFalse

not (非)

条件 (A)not A
TrueFalse
FalseTrue

示例：

print(True and True)    # 输出：True
print(True and False)   # 输出：False
print(False or True)    # 输出：True
print(False or False)   # 输出：False
print(not True)         # 输出：False
print(not False)        # 输出：True

age = 10
has_ticket = True
print(age >= 8 and has_ticket == True) # 10>=8 (True) and True == True (True)  => True and True => True

答案与解析：

答案： True

解析： and 的优先级高于 or。先算 False and True 结果是 False。再算 True or False 结果是 True。

2.

y = 10

答案与解析：

答案： False

答案： True

三、 逻辑运算符的"小秘密"：短路求值和返回值

知识讲解：

Python的 and 和 or 运算符有一个很聪明的特性，叫做短路求值 (short-circuit evaluation)。

对于 and：

  • 如果 and 左边的第一个条件已经是 False 了，那么整个 and 表达式的结果肯定就是 False，Python 就不会再去检查右边的第二个条件了（就像你发现第一把钥匙不对，就不会再试第二把了）。
  • 返回值特性：
    • 如果第一个条件为 False（或者等效于False的值，如 0, "", None），那么 and 运算直接返回第一个条件的值。
    • 如果第一个条件为 True，那么 and 运算会返回第二个条件的值。
  • 示例： 
    print(False and 100)   # 输出：False (因为第一个是False，直接返回它)
print(0 and "Hello") # 输出：0 (0等效于False)
print(True and "Python") # 输出：Python (因为第一个是True，返回第二个的值)
print(5 and "World")    # 输出：World (5等效于True)
# 例如试卷中的 print(6<9 and 10)，6<9是True，所以返回10。

对于 or：

  • 如果 or 左边的第一个条件已经是 True 了，那么整个 or 表达式的结果肯定就是 True，Python 就不会再去检查右边的第二个条件了。
  • 返回值特性：
    • 如果第一个条件为 True（或者等效于True的值），那么 or 运算直接返回第一个条件的值。
    • 如果第一个条件为 False，那么 or 运算会返回第二个条件的值。
  • 示例： 
    print(True or 0)       # 输出：True (因为第一个是True，直接返回它)
print("Hello" or "")   # 输出：Hello ("Hello"等效于True)
print(False or "Python") # 输出：Python (因为第一个是False，返回第二个的值)
print(0 or 20)         # 输出：20 (0等效于False)
# 例如试卷中的 print(17 or 39)，17为True，所以返回17。
# 例如 a = 2 or False and False，先算 False and False 得 False，再算 2 or False，2为True，所以a的值是2。

这个特性有时候可以让代码更高效，或者写出一些简洁的表达。

四、 谁先谁后：逻辑运算符的优先级

知识讲解：

当一个表达式中同时有算术运算符、比较运算符和逻辑运算符时，它们也是有固定的运算顺序的。

通常的优先级顺序（从高到低）：

  1. 括号 ()
  2. 算术运算符 (例如 ** > * / // % > + -)
  3. 比较运算符 (例如 ==, !=, >, <)
  4. 逻辑运算符：
    • not (优先级最高)
    • and (其次)
    • or (优先级最低)

也就是说，电脑会先算数学加减乘除，然后进行大小比较，再进行 not 运算，接着是 and 运算，最后是 or 运算。

如果你不确定优先级，或者想让某一部分先计算，最好的办法就是使用括号 () 来明确指定运算顺序！

例如试卷中提到 if 不属于逻辑运算符，它是条件语句的关键字。

练习题3 (写出 print() 语句的输出结果)：

1. print(5 + 2 > 6 and 10 - 8 == 2)

答案与解析：

答案： True

解析：
1. 算术：5 + 2 = 7，10 - 8 = 2
2. 比较：7 > 6 是 True，2 == 2 是 True
3. 逻辑 and：True and True 是 True

2. print(not 5 > 3 or 2 * 2 == 5)

答案与解析：

答案： False

解析：
1. 算术：2 * 2 = 4
2. 比较：5 > 3 是 True，4 == 5 是 False
3. 逻辑 not：not True 是 False
4. 逻辑 or：False or False 是 False

五、 用逻辑来解决复杂问题

知识讲解：

现在我们有了比较运算符和逻辑运算符，就可以组合出更复杂的条件判断了。

比如，判断一个年份是不是闰年：能被4整除但不能被100整除，或者能被400整除。

year = 2024
is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)
print(is_leap) # 输出：True

或者像之前试卷中判断评选优秀少先队员的条件："语文成绩>=90 and 数学成绩>=90 and 英语成绩>=90"。

编程题1 (在IDLE中完成)：

题目：判断周末出行条件
编写一个程序，假设有两个布尔变量：is_sunny (表示是否晴天) 和 homework_done (表示作业是否完成)。
只有当天气晴朗并且作业也完成了的时候，才适合出去玩。
程序需要获取这两个条件的值（可以让用户输入 "True" 或 "False"，然后你需要想办法转换成真正的布尔值，或者为了简化，直接在代码里给它们赋值 True 或 False 进行测试）。
输出一个布尔值，表示是否适合出去玩。

参考答案 (简化版，直接赋值测试)：
is_sunny = True       # 可以改为 False 测试不同情况
homework_done = False # 可以改为 True 测试不同情况
can_go_out = is_sunny and homework_done
print("是否适合出去玩：", can_go_out)
参考答案 (用户输入版，稍微复杂一点，需要处理字符串到布尔值的转换)：
sunny_input = input("今天天气晴朗吗？(输入 True 或 False): ")
homework_input = input("作业完成了吗？(输入 True 或 False): ")

# 将输入的字符串转换为布尔值 (简单处理)
is_sunny = (sunny_input == "True") 
homework_done = (homework_input == "True")

can_go_out = is_sunny and homework_done
print("是否适合出去玩：", can_go_out)
解析：

程序的核心是 is_sunny and homework_done 这个逻辑表达式，它准确地表达了题目中的条件。

编程题2 (在IDLE中完成)：

题目：数字范围判断器
编写一个程序，提示用户输入一个整数。
判断这个整数是否大于0且小于10，或者是否大于等于100。
输出一个布尔值表示判断结果。

参考答案：
num_str = input("请输入一个整数：")
num = int(num_str)

# 条件1：大于0 且 小于10
condition1 = (num > 0) and (num < 10)
# 条件2：大于等于100
condition2 = (num >= 100)

# 满足条件1 或者 满足条件2
result = condition1 or condition2
print("这个数字是否符合条件：", result)
解析：
  • 程序将复杂的判断拆分成了两个小条件 condition1 和 condition2。
  • 然后用 or 将这两个小条件连接起来，表示只要满足其中一个，最终结果就为真。
  • 注意括号的使用，确保 and 优先于 or （虽然在这里 condition1 和 condition2 是独立计算的，但养成好习惯总是好的）。

今天我们学习了编程中非常重要的"逻辑判断工具"——逻辑运算符 and、or 和 not。它们就像我们说话时的"并且"、"或者"和"不是"，可以帮助我们把简单的判断组合成更复杂的条件。我们还了解了它们运算的先后顺序以及一些"小秘密"（短路求值）。掌握好逻辑运算，我们就能让程序根据不同的情况做出更智能的反应啦！
